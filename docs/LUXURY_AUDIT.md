# Creative Audit: Cinematic Luxury Overhaul

## Текущее состояние — критические проблемы

### 1. **First impression = ноль**
- Плоский `bg-background` (#FAFAFA), никакой атмосферы
- Header появляется на пустом фоне без wow-момента
- Нет layered depth, нет light, нет motion
- Первый экран — это сразу search panel: утилитарно, не эмоционально

### 2. **Нет storytelling**
- Секции: Search → Brands → Recommendations — механический список
- Нет narrative flow, нет reveal, нет «пути» пользователя
- Визуально все блоки одинаковые по тону и композиции
- Никакого scroll-driven narrative

### 3. **Визуальная система — шаблонная**
- Inter везде — надёжно, но нулевая brand identity
- Light theme dominant — не luxury automotive
- Teal palette хорош, но применён плоско
- Нет carbon, нет premium materials, нет tech surfaces

### 4. **Карточки — functional, не premium**
- Белый фон, border, shadow-sm
- Нет эмоциональной работы с изображением (overlay, vignette, glow)
- Бейджи читаемые, но не luxury
- Price typography стандартная
- Trust signals слабые визуально

### 5. **Listing page — transactional, не luxury**
- Hero слабый (title + subtitle)
- Gallery без cinematic frame
- Price/VIN/Seller — ровные карточки, нет premium differentiation
- Нет ощущения luxury product page

### 6. **Motion — почти отсутствует**
- Staggered reveal только в Recommendations
- Нет scroll transitions
- Нет cursor interaction
- Нет parallax, нет layered motion

### 7. **Background — мёртвый**
- Один цвет
- Нет атмосферы, градиентов, света
- Нет cursor-reactive layer
- Нет carbon, mesh, gloss

### 8. **Trust UI — формальный**
- Trust bar в footer — текст + точки
- VIN block — иконка + список
- Seller card — стандартная карточка
- Нет визуального «веса» доверия

### 9. **Dark theme — вторичен**
- Токены есть, но светлая тема доминирует
- Dark — просто инвертированные цвета
- Нет luxury dark surfaces (carbon, deep blacks, accent glows)

### 10. **Mobile / Reduced motion — не учтены**
- Анимации без fallback
- Нет @media (prefers-reduced-motion: reduce)

---

## Вердикт

Сайт выглядит как качественный MVP маркетплейса, но не как premium automotive platform. Нет эмоционального крючка, нет luxury identity, нет cinematic storytelling. Визуал функциональный, но не aspirational.
